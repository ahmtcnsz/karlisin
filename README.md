# Karlısın - Finansal Karar Destek Platformu

Karlısın, yatırımcıların ve e-ticaret satıcılarının finansal verilerini analiz etmelerine, pasif gelir stratejileri oluşturmalarına ve kârlılıklarını optimize etmelerine yardımcı olan yapay zeka destekli bir platformdur.

## 🚀 Öne Çıkan Özellikler

- **2026 Temettü Takvimi:** BİST ve Global borsalar için güncel temettü ödeme tarihleri ve verim oranları.
- **AI Destekli Tahmin Motoru:** Google Gemini 1.5 Flash kullanarak şirket bilançoları üzerinden gelecek dönem temettü tahminleri.
- **Maaş Hesaplama Robotu:** 2026 vergi dilimlerine uyumlu brütten nete maaş ve yan hak hesaplayıcı.
- **Pazar Yeri Kâr Hesaplayıcı:** Trendyol, Hepsiburada ve Amazon satıcıları için komisyon, kargo ve reklam maliyetlerini içeren net kâr analizi.
- **Piyasanın Nabzı:** Global piyasa verileri ve makroekonomik göstergelerin gerçek zamanlı takibi.

## 🛠️ Teknoloji Yığını

- **Frontend:** React 19, Vite, Tailwind CSS v4, Motion (Animations)
- **Backend:** Node.js (Express)
- **Database & Auth:** Firebase (Firestore & Auth)
- **AI Integration:** Google Gemini API
- **Icons:** Lucide React
- **Charts:** Recharts

## 📦 Kurulum ve Çalıştırma

### Gereksinimler
- Node.js >= 20.0.0
- npm or yarn

### Adımlar

1. Projeyi klonlayın:
   ```bash
   git clone https://github.com/kullaniciadi/karlisin.git
   cd karlisin
   ```

2. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```

3. Gerekli ortam değişkenlerini `.env` dosyasına ekleyin ( `.env.example` dosyasını referans alabilirsiniz ).

4. Geliştirme sunucusunu başlatın:
   ```bash
   npm run dev
   ```

## 📈 SEO ve Görünürlük

Bu proje, özellikle **"2026 temettü takvimi"**, **"temettü verimi hesaplama"** ve **"maaş hesaplama 2026"** gibi "long-tail" anahtar kelimelere odaklanmış SEO stratejileri ile donatılmıştır. Tüm sayfalar `react-helmet-async` ile dinamik meta etiketlerine ve yapılandırılmış sitemap verilerine sahiptir.
